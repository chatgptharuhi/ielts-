## Issue①: 単語テスト内の重複出現を防止

### 要望
「同じ単語が1セッション内に2回出現してしまう」

### 根本原因
- セッション状態を追跡していなかった
- `pickNextWord()` がセッション履歴を参照していない

### 実装

**変更点:**
```javascript
// STATE に追加
let wordSessionHistory = new Set();  // セッション内で既に出題した単語

// startWordTest() で初期化
wordSessionHistory = new Set();

// pickNextWord() でフィルタ
pool = pool.filter(w => !wordSessionHistory.has(w.word));

// nextWord() で記録
wordSessionHistory.add(currentWord.word);
```

**効果:**
- 1セッション内で同じ単語は最大1回のみ出題
- セッション間では履歴がリセット（新しいセッションで再出題可能）

## Issue②: 選択肢の品詞を統一

### 要望
「選択肢が同じ品詞でそろっていない（例：動詞の問題に名詞の選択肢）」

### 根本原因
```javascript
// 修正前
const pool = [...samePos, ...diffPos].slice(0, 5);  // 混在
```

### 実装

**修正:**
```javascript
// 修正後
const samePos = shuffle(choiceBase.filter(w => w.word !== word.word && w.pos === word.pos));
const wrong = samePos.slice(0, 5).map(w => ({ meaning: w.meaning, isCorrect: false }));
```

**仕様:**
- 選択肢 = 同じ品詞の単語のみ
- 同じ品詞が 5個未満 → その数だけ表示（選択肢数が 4個になる場合もある）

**効果:**
- 単語追加が進むにつれて、自動的に選択肢が充実
- 品詞マッチの学習効果向上
